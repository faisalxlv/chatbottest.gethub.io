نظرة عامة

روبوت المحادثة لمتجر الدخيل للعود هو مساعد افتراضي ذكي مصمم لتحسين تجربة العملاء وتقديم دعم فوري لهم. يهدف الروبوت إلى مساعدة العملاء في الاستعلام عن المنتجات، والمساعدة في عملية الشراء، وتقديم معلومات عن سياسات المتجر، والتعامل مع المشاكل والشكاوى.

يستخدم الروبوت تقنيات الذكاء الاصطناعي ومعالجة اللغة الطبيعية لفهم استفسارات العملاء والرد عليها بشكل دقيق وفعال. كما يتميز بقدرته على التعلم والتحسن مع مرور الوقت بناءً على تفاعلات المستخدمين.

ملاحظة: هذا التوثيق مخصص للمطورين والمسؤولين التقنيين الذين يرغبون في فهم كيفية عمل الروبوت وتخصيصه وتكامله مع أنظمة أخرى.

البدء السريع

لبدء استخدام روبوت المحادثة، اتبع الخطوات التالية:

1. متطلبات النظام

  • متطلبات الخادم:
    • نظام التشغيل: Linux (Ubuntu 20.04 LTS أو أحدث)
    • وحدة المعالجة المركزية: 4 نوى على الأقل
    • ذاكرة الوصول العشوائي: 8 جيجابايت على الأقل
    • مساحة التخزين: 50 جيجابايت على الأقل (SSD مفضل)
  • البرمجيات المطلوبة:
    • Node.js (v14+)
    • Python (v3.8+)
    • MongoDB (v4.4+)
    • Redis (v6.0+)

2. التثبيت

يمكن تثبيت روبوت المحادثة باستخدام Docker أو من المصدر مباشرة.

التثبيت باستخدام Docker

# سحب صورة Docker docker pull aldakheeloud/chatbot:latest # تشغيل الحاوية docker run -d -p 3000:3000 -p 5000:5000 \ -e MONGODB_URI=mongodb://mongodb:27017/aldakheeloud-chatbot \ -e REDIS_HOST=redis \ -e REDIS_PORT=6379 \ --name aldakheeloud-chatbot \ aldakheeloud/chatbot:latest

التثبيت من المصدر

# استنساخ المستودع git clone https://github.com/aldakheeloud/chatbot.git cd chatbot # تثبيت اعتماديات Node.js cd frontend npm install cd ../backend npm install # تثبيت اعتماديات Python cd ../conversation-engine pip3 install -r requirements.txt # تشغيل التطبيق cd .. docker-compose up -d

3. التكوين الأساسي

بعد التثبيت، يمكنك تكوين الروبوت من خلال تعديل ملف config.js في المجلد الرئيسي:

// config.js module.exports = { // إعدادات قاعدة البيانات database: { mongodb: { uri: process.env.MONGODB_URI || 'mongodb://localhost:27017/aldakheeloud-chatbot' }, redis: { host: process.env.REDIS_HOST || 'localhost', port: process.env.REDIS_PORT || 6379 } }, // إعدادات الروبوت chatbot: { name: 'مساعد الدخيل للعود', initialMessage: 'مرحباً بك في متجر الدخيل للعود! كيف يمكنني مساعدتك اليوم؟', language: 'ar', suggestedQuestions: [ 'ما هي المنتجات المتوفرة؟', 'ما هي سياسة الاسترجاع؟', 'كم تستغرق عملية التوصيل؟', 'ما هي طرق الدفع المتاحة؟' ] }, // إعدادات الخادم server: { port: process.env.PORT || 3000, apiPath: '/api' } };

الميزات الرئيسية

1. معالجة اللغة الطبيعية

يستخدم الروبوت تقنيات معالجة اللغة الطبيعية (NLP) لفهم استفسارات العملاء بشكل دقيق. يتضمن ذلك:

  • تحليل النية (Intent Analysis): تحديد الغرض من استفسار العميل.
  • استخراج الكيانات (Entity Extraction): استخراج المعلومات المهمة مثل أسماء المنتجات والفئات.
  • تحليل المشاعر (Sentiment Analysis): تحديد مشاعر العميل للاستجابة بشكل مناسب.

2. قاعدة المعرفة

يعتمد الروبوت على قاعدة معرفة شاملة تحتوي على معلومات عن:

  • المنتجات وفئاتها
  • الأسعار والعروض
  • سياسات المتجر (الشحن، الاسترجاع، الخصوصية)
  • الأسئلة الشائعة وإجاباتها

3. تدفقات المحادثة

يستخدم الروبوت تدفقات محادثة مصممة مسبقاً للتعامل مع مختلف سيناريوهات التفاعل مع العملاء، مثل:

  • الاستعلام عن المنتجات
  • المساعدة في عملية الشراء
  • الاستعلام عن سياسات المتجر
  • التعامل مع المشاكل والشكاوى

4. التكامل مع أنظمة المتجر

يمكن تكامل الروبوت مع أنظمة المتجر المختلفة، مثل:

  • نظام إدارة المحتوى
  • نظام إدارة المخزون
  • نظام إدارة علاقات العملاء (CRM)
  • نظام تتبع الطلبات

تدفقات المحادثة

تدفقات المحادثة هي مسارات محددة مسبقاً للتفاعل بين الروبوت والعملاء. تساعد هذه التدفقات الروبوت على توجيه المحادثة بطريقة منظمة ومفيدة.

1. تدفق الترحيب والتعريف

المتحدث الرسالة

المستخدم ما هي المنتجات التي تقدمونها؟
الروبوت متجر الدخيل للعود متخصص في بيع أجود أنواع العطور ومنتجات العود والبخور. نقدم مجموعة متنوعة من المنتجات تشمل:
- العطور الفاخرة
- العود والبخور بأنواعه
- الزيوت العطرية والمخلطات
- عطور الشعر ومعطرات الجسم
- معطرات المنزل
- مجموعات الهدايا والتوزيعات

هل تبحث عن فئة معينة من هذه المنتجات؟

التخصيص

يمكن تخصيص روبوت المحادثة بعدة طرق لتلبية احتياجات متجرك.

1. تخصيص المظهر

يمكنك تخصيص مظهر الروبوت من خلال تعديل ملف CSS:

/* تخصيص ألوان الروبوت */ :root { --primary-color: #8b5a2b; /* اللون الرئيسي */ --secondary-color: #d4af37; /* اللون الثانوي */ --dark-color: #333; /* لون النص الداكن */ --light-color: #f4f4f4; /* لون الخلفية الفاتح */ } /* تخصيص حجم الروبوت */ .chatbot-widget { width: 350px; /* عرض الروبوت */ height: 500px; /* ارتفاع الروبوت */ } /* تخصيص موضع الروبوت */ .chatbot-widget { bottom: 20px; /* المسافة من أسفل الصفحة */ left: 20px; /* المسافة من يسار الصفحة */ }

2. تخصيص الردود

يمكنك تخصيص ردود الروبوت من خلال تعديل ملف الردود:

// responses.js const responses = { "مرحبا": "مرحباً بك في متجر الدخيل للعود! كيف يمكنني مساعدتك اليوم؟", "السلام عليكم": "وعليكم السلام ورحمة الله وبركاته! أهلاً بك في متجر الدخيل للعود. كيف يمكنني خدمتك؟", "ما هي المنتجات المتوفرة": "متجر الدخيل للعود متخصص في بيع أجود أنواع العطور ومنتجات العود والبخور. نقدم مجموعة متنوعة من المنتجات تشمل:\n- العطور الفاخرة\n- العود والبخور بأنواعه\n- الزيوت العطرية والمخلطات\n- عطور الشعر ومعطرات الجسم\n- معطرات المنزل\n- مجموعات الهدايا والتوزيعات\n\nهل تبحث عن فئة معينة من هذه المنتجات؟", // المزيد من الردود... }; module.exports = responses;

الأسئلة الشائعة

1. هل يمكن استخدام الروبوت على الهواتف المحمولة؟

نعم، الروبوت متوافق تماماً مع الهواتف المحمولة والأجهزة اللوحية، ويتكيف تلقائياً مع حجم الشاشة.

2. هل يدعم الروبوت لغات أخرى غير العربية؟

حالياً، الروبوت مصمم للعمل باللغة العربية بشكل أساسي. يمكن إضافة دعم للغات أخرى من خلال تخصيص الروبوت وإضافة نماذج لغوية إضافية.

3. كيف يمكنني تحديث قاعدة المعرفة؟

يمكنك تحديث قاعدة المعرفة من خلال تعديل ملفات JSON في المجلد knowledge-base أو من خلال واجهة الإدارة إذا كانت متاحة.

4. هل يمكن تكامل الروبوت مع نظام إدارة علاقات العملاء (CRM)؟

نعم، يمكن تكامل الروبوت مع أنظمة CRM الشائعة مثل Salesforce وHubSpot وZoho CRM من خلال واجهة برمجة التطبيقات (API).

5. كيف يمكنني مراقبة أداء الروبوت؟

يوفر الروبوت لوحة تحكم للتحليلات تعرض مقاييس مثل عدد المحادثات، ومعدل الرضا، والأسئلة الأكثر شيوعاً، ونقاط الفشل في المحادثات.
